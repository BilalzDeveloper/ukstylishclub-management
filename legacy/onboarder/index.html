▶ How to get your Google OAuth Client ID (one-time)
1. Go to console.cloud.google.com
2. Create a new project (e.g. "UKSC Onboarder")
3. Go to APIs & Services → Enable APIs → enable Google Drive API
4. Go to APIs & Services → OAuth consent screen
   • User type: External → Create
   • App name: UKSC Onboarder, your email, save
   • Scopes: add ../auth/drive → save
   • Test users: add bilalshami@gmail.com
5. Go to Credentials → Create Credentials → OAuth Client ID
   • Type: Web application
   • Authorised JavaScript origins: https://bilalzdeveloper.github.io
   • Authorised redirect URIs: https://bilalzdeveloper.github.io/uksc-onboarder/
6. Copy the Client ID and paste it above
7. Click "Sign in with Google" — done forever ✓
